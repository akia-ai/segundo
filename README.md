# Hotel Segundo

Demo website for Hotel Segundo, a fictional luxury boutique hotel in San Diego, CA.

## Pages

- `index.html` - Main hotel landing page with booking widget
- `booking.html` - Fake booking engine that displays arrival/departure dates

## Usage

Open `index.html` in a browser. The booking page accepts URL parameters:

```
booking.html?arrival=2024-03-15&departure=2024-03-18
```

## Placeholder Images

All images use placehold.co placeholders. Replace the image URLs with real hotel photos as needed.
